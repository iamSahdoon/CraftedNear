# CraftedNear

CraftedNear is a full-stack e-commerce web application developed as a Degree Research Project. The platform enables customers to discover nearby sellers and allows sellers to create and manage their own digital storefronts.

## Project Overview

CraftedNear was designed to improve accessibility to local sellers by providing a centralized digital marketplace. The system supports role-based access control and includes engagement-focused features to enhance user interaction and seller visibility.

## Key Features

- Seller registration and digital storefront creation  
- Customer discovery of nearby sellers  
- Role-based authentication and authorization using JWT  
- Customer reviews and rating system  
- Exclusive offers for authenticated users  
- Favorite store management  
- Gamification system including:
  - Points
  - Badges
  - Titles
  - Leaderboards  
- Media upload and management using Cloudinary  

## User Feedback

Based on collected feedback, 84% of users confirmed improved usability, better access to local stores, and enhanced overall interaction with the platform.

## Technology Stack

**Frontend**
- React
- Context API

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**Authentication**
- JSON Web Tokens (JWT)

**Media Management**
- Cloudinary

## System Architecture

The application follows a MERN stack architecture with a REST-based communication model between the frontend and backend services.
